# Market Pathshala (React + Vite + Tailwind)

## Quick Start
```
npm install
npm run dev
```
Then open http://localhost:5173/market-pathshala

## Build & Deploy to GitHub Pages
- Set repository to `market-pathshala` under your account.
- In `vite.config.js`, `base` is `/market-pathshala/` (works with GitHub Pages).
```
npm run build
```
Push the `dist/` to gh-pages or enable Pages on `main` with `/(root)`.
